## 1. 显示字段

| 页面元素 | 字段名 | 来源类型 | 具体来源 | 备注 |
|---------|--------|---------|---------|------|
| 视频缩略图 | thumbnail_url | 📥存储 | 爆款视频.thumbnail_url | 底部渐变遮罩，加载失败显示灰色占位图 |
| 视频时长标签 | video_duration | 📥存储 | 爆款视频.video_duration | 格式 MM:SS |
| 文案状态角标 | transcript_status | 📥存储 | 爆款视频.transcript_status | 映射：已提取/待提取/提取失败/提取中 |
| 视频标题 | video_title | 📥存储 | 爆款视频.title | 最多2行省略，>40字悬停tooltip |
| 播放量 | play_count | 📥存储 | 爆款视频.play_count | ≥10000显示"X.X万"；为0显示"0" |
| 点赞数 | like_count | 📥存储 | 爆款视频.like_count | 同上 |
| 评论数 | comment_count | 📥存储 | 爆款视频.comment_count | 同上 |
| 收藏数 | collect_count | 📥存储 | 爆款视频.collect_count | 同上 |
| 互动率 | interaction_rate | 📊计算 | (爆款视频.like_count + 爆款视频.comment_count + 爆款视频.collect_count) / 爆款视频.play_count * 100 | 播放量为0显示"--"；格式保留1位小数 |
| 来源账号 | account_name | 📥存储 | 对标账号.account_name | 通过爆款视频.account_id JOIN |
| 平台标签 | platform | 📥存储 | 爆款视频.platform | 映射：xiaohongshu→小红书, douyin→抖音, weixin→视频号 |
| 发布时间 | published_at | 📥存储 | 爆款视频.published_at | 格式 MM-DD HH:mm |
| 视频总数 | video_total | 📊计算 | 列表API响应中的 total 字段 | 标题旁显示"共 N 条视频" |
| 用户头像 | avatar_url | 📥存储 | 用户表.avatar_url | Header装饰 |
| 已选计数 | selected_count | 📊计算 | 前端状态 selected_ids.length | 批量操作栏 |

## 2. 操作与API

### 2.1 查询类操作

| 操作名称 | HTTP方法 | API端点 | 请求参数 | 响应格式 | 变更数据 | 刷新范围 |
|---------|---------|---------|---------|---------|---------|---------|
| 获取视频列表 | GET | /api/v1/viral-videos | page(int, 默认1), page_size(int, 默认12), platform(string, 可选: xiaohongshu/douyin/weixin), account_id(int, 可选), time_range(string, 可选: 7d/30d/all), min_engagement(int, 可选), sort_by(string, 默认interaction_rate: interaction_rate/play_count/published_at/collect_count), order(string, 默认desc: asc/desc), keyword(string, 可选, 按标题模糊搜索) | { total: number, page: number, page_size: number, items: [{ id, account_id, account_name, platform, thumbnail_url, video_duration, title, play_count, like_count, comment_count, collect_count, interaction_rate, transcript_status, published_at }] } | 无 | 视频卡片网格区 |

### 2.2 操作类操作

| 操作名称 | HTTP方法 | API端点 | 请求参数 | 响应格式 | 变更数据 | 刷新范围 |
|---------|---------|---------|---------|---------|---------|---------|
| 加入分析 | POST | /api/v1/viral-videos/analyze-batch | { video_ids: number[] (1-5个) } | { success: boolean, task_id: number } | 选题工作台数据 | 无（跳转选题工作台） |
| 导出数据 | POST | /api/v1/viral-videos/export | { video_ids: number[], format: string (默认xlsx) } | 二进制文件流 (application/vnd.openxmlformats-officedocument.spreadsheetml.sheet) | 无 | 无（触发浏览器下载） |

### 2.3 辅助查询

| 操作名称 | HTTP方法 | API端点 | 请求参数 | 响应格式 | 变更数据 | 刷新范围 |
|---------|---------|---------|---------|---------|---------|---------|
| 获取对标账号列表 | GET | /api/v1/benchmark-accounts?brief=true | 无 | { items: [{ id, account_name, platform }] } | 无 | 筛选栏对标账号下拉 |

## 3. 前端状态

| 数据 | 存储位置 | 初始化时机 | 更新触发 |
|------|---------|-----------|---------|
| 视频列表 | 组件状态 (ref) | 页面加载时调用 GET /api/v1/viral-videos | 筛选/排序/搜索/翻页变更 |
| 分页信息 (page, page_size, total) | 组件状态 (ref) | 页面加载时 | 加载更多 / 筛选变更 |
| 筛选条件 (platform, account_id, time_range, min_engagement) | 组件状态 (reactive) | 默认值：全部/全部/近7天/空 | 筛选控件操作 |
| 排序条件 (sort_by, order) | 组件状态 (ref) | 默认值：interaction_rate/desc | 排序下拉操作 |
| 搜索关键词 (keyword) | 组件状态 (ref) | 默认值：空 | 搜索框输入（防抖500ms） |
| 视图模式 (view_mode) | 本地存储 (localStorage) | 从localStorage读取，默认grid | 视图切换按钮点击 |
| 已选视频ID列表 (selected_ids) | 组件状态 (ref) | 默认值：[] | 勾选框点击/全选/取消选择 |
| 加载状态 (loading) | 组件状态 (ref) | 默认值：true | API请求开始/结束 |
| 加载更多状态 (loading_more) | 组件状态 (ref) | 默认值：false | 滚动到底部触发/数据返回 |
| 是否全部加载完 (all_loaded) | 组件状态 (computed) | 计算属性：当前数据量 >= total | 分页数据更新 |
| 对标账号选项列表 (account_options) | 组件状态 (ref) | 页面加载时调用 GET /api/v1/benchmark-accounts?brief=true | 无（只加载一次） |

## 4. 状态流转

### 4.1 文案提取状态 (transcript_status)

```
待提取 ──(触发提取)──→ 提取中 ──(成功)──→ 已提取
                          │
                          └──(失败)──→ 提取失败 ──(重新提取)──→ 提取中
```

### 4.2 页面加载状态

```
初始加载中 ──(成功)──→ 已加载（展示卡片网格）
     │
     └──(失败)──→ 错误状态 ──(重试)──→ 初始加载中
```

### 4.3 批量选择状态

```
未选择 ──(勾选1个)──→ 已选择(显示批量操作栏) ──(继续勾选/全选)──→ 已选择(更新计数)
                           │                                          │
                           │←──(取消选择)──────────────────────────────│
                           │
                           └──(加入分析/导出)──→ 操作中(按钮loading) ──(完成)──→ 未选择
```

## 5. 跨页面参数

| 关联页面 | 关联方式 | 传出参数 | 接收参数 |
|---------|---------|---------|---------|
| 视频详情（含文案） | 点击视频卡片跳转 (router.push) | video_ids[]: number[] (即使单个也用数组格式) | 通过路由参数 $route.query.video_ids[] 接收 |
| 选题工作台 | "加入分析"按钮跳转 (router.push) | video_ids[]: number[] (1-5个) | 通过路由参数或API关联 |
| 对标账号管理 | 来源账号名称点击跳转 (router.push) | account_id: number | 通过路由参数 $route.query.account_id 接收 |
| 数据采集任务 | 空状态"去采集"按钮跳转 | 无 | 接收 task_id, account_id（预填筛选条件） |
| 仪表盘 | Sidebar导航 | 无 | 无 |

### 路由定义

- 页面路径：`/videos`
- 详情跳转：`/videos/detail?video_ids[]={id}`
- 选题跳转：`/workspace?video_ids[]={id1}&video_ids[]={id2}&...`
- 账号跳转：`/accounts?account_id={id}`

## 6. 待确认项

| # | 待确认内容 | 影响范围 | 建议方案 |
|---|-----------|---------|---------|
| 1 | 导出文件的字段范围和格式 | 导出功能 | 导出Excel含：标题、账号、平台、播放/点赞/评论/收藏数、互动率、发布时间、文案状态 |
| 2 | 列表视图模式下的卡片布局 | UI展示 | 列表模式使用单行横向卡片，左侧缩略图(160x90)，右侧信息横向排列 |
| 3 | "加入分析"后是否留在当前页 | 交互流程 | 建议：留在当前页，顶部显示成功提示；同时可在顶部提供"前往选题工作台"链接 |
| 4 | 视频时长字段是否在数据库中存储 | 数据模型 | 需确认爆款视频表是否有 video_duration 字段；若无需新增 |
| 5 | 互动量筛选阈值 min_engagement 的计算口径 | 筛选逻辑 | 互动量 = 点赞+评论+收藏（不含播放量），与互动率分母不同 |

| 影响项 | 影响描述 | 影响级别 |

| API | 新增 GET /api/v1/benchmark-accounts?brief=true 对标账号简要列表接口（轻量） | 低 |

| 前端 | 新增视频卡片组件（网格/列表双模式） | 高 |

| 前端 | 新增批量操作栏组件 | 低 |
| 跨页面 | 需与视频详情页约定 video_ids[] 数组传参格式（即使单个也用数组） | 低 |
| 跨页面 | 需与选题工作台约定 video_ids[] 数组传参格式 | 低 |
