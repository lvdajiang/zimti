## 2. 操作与API

| 操作名称 | HTTP方法 | API端点 | 请求参数 | 响应格式 | 变更数据 | 刷新范围 |
|---------|---------|---------|---------|---------|---------|---------|
| 获取素材列表 | GET | /api/v1/materials | type(可选), sort_by(默认created_at), keyword(可选), page(默认1), page_size(默认20) | { items: Material[], total: number, stats: {total, image, video, music, sfx, map_animation} } | 无 | 素材网格+统计 |
| 上传素材 | POST | /api/v1/materials | multipart/form-data: files[](多文件), type(自动识别或手动指定) | { items: Material[] } | materials表新增记录 | 素材网格+统计 |
| AI生成素材 | POST | /api/v1/materials/generate | { type: "图片"/"视频", description: string } | { task_id: string, material: Material(预览) } | materials表新增记录 | 素材网格+统计 |
| 轮询AI生成状态 | GET | /api/v1/materials/generate/{task_id}/status | 无 | { status: "generating"/"completed"/"failed", progress: number, material?: Material } | 无 | AI生成进度条 |
| 确认AI生成素材 | POST | /api/v1/materials/generate/{task_id}/confirm | 无 | { material: Material } | materials表更新status为confirmed | 素材网格+统计 |
| 外部导入素材(Pexels搜索) | GET | /api/v1/materials/pexels-search | keyword, type(图片/视频), page, page_size | { items: PexelsItem[], total: number } | 无 | 外部导入弹窗列表 |
| 外部导入素材(确认导入) | POST | /api/v1/materials/pexels-import | { pexels_id: string, type: string, source_url: string } | { material: Material } | materials表新增记录 | 素材网格+统计 |
| 编辑素材信息 | PUT | /api/v1/materials/:id | { name?, tags?[], copyright_status?, source? } | { material: Material } | materials表对应字段 | 对应素材卡片 |
| 删除素材 | DELETE | /api/v1/materials/:id | 无 | { success: boolean } | materials表删除记录 | 素材网格+统计 |
| 检查素材引用 | GET | /api/v1/materials/:id/references | 无 | { count: number, videos: {id, title}[] } | 无 | 删除按钮禁用状态 |
| 下载素材 | GET | /api/v1/materials/:id/download | 无 | 二进制文件流(download) | 无 | 无(浏览器触发下载) |
| 获取素材统计 | GET | /api/v1/materials/stats | 无 | { total, image, video, music, sfx, map_animation } | 无 | 标题行统计 |

## 4. 状态流转

### 4.1 素材上传状态流转

```
选择文件 → 验证(格式+大小) → 排队等待 → 上传中(progress 0-100%) → 上传成功 → 刷新列表
                                                          ↘ 上传失败 → 可重试 → 重新上传
```

- 格式验证：仅允许 JPG/PNG/MP4/MP3/WAV
- 大小验证：单文件不超过500MB
- 同名处理：后端自动追加编号，不覆盖
- 上传模式：串行排队，不支持并行

### 4.2 AI生成素材状态流转

```
输入描述 → 点击生成 → 后端创建任务(generating) → 轮询进度(0-100%) → 生成完成(preview_url)
                    ↘ 超时(>60秒) → "生成时间较长，请稍候..." → 继续轮询
                                                                        ↘ 生成失败 → 提示重试
生成完成 → 用户确认"添加到素材库" → 素材正式入库 → 刷新列表
```

### 4.3 素材删除状态流转

```
点击删除 → 检查引用状态 → 有引用 → 禁用删除，提示"该素材已被N个视频引用"
                         → 无引用 → 弹出确认弹窗 → 确认删除 → 后端删除 → 卡片fadeOut → 刷新列表+统计
```

### 4.4 素材编辑状态流转

```
点击编辑 → 弹窗加载当前素材信息 → 修改字段 → 点击保存 → 后端更新 → 关闭弹窗 → 刷新对应卡片
```

### 4.5 筛选/搜索状态流转

```
改变筛选条件(type_filter/sort_by/search_keyword) → 重置current_page=1 → 重新请求GET /materials → 更新列表
```

## 6. 待确认项

| # | 待确认项 | 影响范围 | 建议 |
|---|---------|---------|------|
| 1 | AI生成素材的具体调用模型（DALL-E/MidJourney/自训练模型） | AI生成API的请求/响应格式 | 后端抽象统一接口，前端不感知底层模型 |
| 2 | Pexels API的调用频率限制和认证方式 | 外部导入功能的可用性 | 后端代理Pexels请求，前端不直接调用 |
| 3 | 素材文件的存储方案（OSS/S3/本地） | 文件上传/下载API的实现 | 建议使用OSS，上传走后端签名URL |
| 4 | 音频波形图是存储还是实时生成 | thumbnail_url对音频类型的处理 | 建议上传时后端生成波形图并存储 |
| 5 | 批量上传的最大文件数量限制 | 上传弹窗的文件列表展示 | 建议10个/批 |
| 6 | 素材标签是否需要全局标签库（标准化标签） | 标签输入交互和搜索 | 建议第一阶段自由输入，后续迭代标签库 |

## 7. 对现有系统的影响

| 影响项 | 影响描述 | 应对方案 |
|-------|---------|---------|
| 数据库新增表 | materials表、video_materials关联表 | 需新建迁移文件，material_id在关联表中需建索引 |
| 文件存储依赖 | 需要文件存储服务支持（OSS/S3） | 架构设计时确定存储方案 |
| AI服务依赖 | AI生成素材依赖外部AI模型服务 | 后端封装统一接口，增加降级策略 |
| Pexels外部API | 外部导入依赖Pexels API可用性 | 后端代理+缓存，增加超时和降级处理 |
| 素材与视频关联 | video_materials多对多关联表 | 渲染时批量查询素材，删除时检查引用 |
| use_count计数 | 素材被视频引用时需更新use_count | 视频创建/删除时触发计数更新（建议用触发器或应用层事件） |
