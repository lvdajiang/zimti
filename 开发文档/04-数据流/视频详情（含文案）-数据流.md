## 2. 操作与API

### 2.1 查询类操作

| 操作名称 | HTTP方法 | API端点 | 请求参数 | 响应格式 | 变更数据 | 刷新范围 |
|---------|---------|---------|---------|---------|---------|---------|
| 获取视频详情 | GET | /api/v1/viral-videos/:id | id(路径参数, number) | { id, account_id, account_name, platform, video_url, thumbnail_url, title, play_count, like_count, comment_count, collect_count, share_count, interaction_rate, published_at, transcript_status, transcript: { id, transcript_text, extraction_method, extracted_at, status } | null, analysis: { id, hook_type, content_structure, conversion_method, highlight_tags, analyzed_at } | null } | 无 | 全页面 |

### 2.2 操作类操作

| 操作名称 | HTTP方法 | API端点 | 请求参数 | 响应格式 | 变更数据 | 刷新范围 |
|---------|---------|---------|---------|---------|---------|---------|
| 复制文案 | 无（纯前端操作） | 无 | 无 | 无（使用 navigator.clipboard.writeText） | 无 | 无 |
| 重新提取文案 | POST | /api/v1/viral-videos/:id/transcript/extract | id(路径参数, number) | { success: boolean, task_id: string } | 文案提取表新增/更新记录 | 文案区域（切换为骨架屏） |
| AI拆解 | POST | /api/v1/viral-videos/:id/analyze | id(路径参数, number) | { success: boolean, analysis_id: number } | 爆款拆解表新增记录 | 爆款拆解区（切换为骨架屏） |
| 保存文案编辑 | PUT | /api/v1/viral-videos/:id/transcript | id(路径参数, number), transcript_text(string) | { success: boolean, updated_at: string } | 文案提取表.transcript_text | 文案区域 |
| 用作对标（加入选题） | POST | /api/v1/topic-proposals/add-videos | { video_ids: number[], video_titles: string[] } | { success: boolean, added_count: number } | 选题素材关联表新增记录 | 无（跳转选题工作台） |
| 下载视频 | GET | /api/v1/viral-videos/:id/download | id(路径参数, number) | 二进制文件流 (video/mp4) | 无 | 无（触发浏览器下载） |

## 3. 前端状态

| 数据 | 存储位置 | 初始化时机 | 更新触发 |
|------|---------|-----------|---------|
| 视频详情 (video_detail) | 组件状态 (ref) | 页面加载时调用 GET /api/v1/viral-videos/:id | 重新提取/AI拆解/保存编辑完成后刷新 |
| 文案提取状态 (transcript_status) | 组件状态 (computed) | 从 video_detail 计算 | 提取操作完成 |
| 文案编辑模式 (is_editing) | 组件状态 (ref) | 默认值：false | 编辑/取消按钮切换 |
| 编辑中文案内容 (editing_text) | 组件状态 (ref) | 进入编辑时复制 transcript_text | 编辑文本框输入 |
| AI拆解状态 (analysis_status) | 组件状态 (computed) | 从 video_detail.analysis 计算：有analyzed_at→已拆解，无→未拆解 | AI拆解操作完成 |
| 拆解加载中 (analyzing) | 组件状态 (ref) | 默认值：false | AI拆解按钮点击/完成 |
| 提取加载中 (extracting) | 组件状态 (ref) | 默认值：false | 重新提取按钮点击/完成 |
| 保存加载中 (saving) | 组件状态 (ref) | 默认值：false | 保存按钮点击/完成 |
| 页面加载状态 (loading) | 组件状态 (ref) | 默认值：true | API请求开始/结束 |
| 页面错误状态 (error) | 组件状态 (ref) | 默认值：null | API请求失败 |
| 未保存提示 (has_unsaved_changes) | 组件状态 (computed) | is_editing && editing_text !== transcript_text | 编辑文本框输入 |
| 提取任务轮询 (poll_timer) | 组件状态 (ref) | 触发提取后启动 | 3秒轮询一次，完成后停止 |

## 4. 状态流转

### 4.1 文案提取状态 (transcript_status)

```
待提取 ──(点击提取文案/重新提取)──→ 提取中
                                        │
                                        ├──(成功)──→ 已提取（显示完整文案）
                                        │
                                        └──(失败)──→ 提取失败（显示错误+重新提取按钮）
```

### 4.2 文案编辑状态

```
只读态 ──(点击编辑)──→ 编辑态 ──(点击保存)──→ 保存中 ──(成功)──→ 只读态(更新内容)
                            │                      │
                            │                      └──(失败)──→ 编辑态(保留修改)
                            │
                            └──(点击取消)──→ 只读态(放弃修改)
```

### 4.3 AI拆解状态

```
未拆解（引导卡片）──(点击AI拆解)──→ 分析中（骨架屏 3-8s）
                                       │
                                       ├──(成功)──→ 已拆解（显示拆解结果）
                                       │
                                       └──(失败)──→ 拆解失败（错误提示+重试按钮）
```

### 4.4 页面加载状态

```
初始加载中（骨架屏）──(成功)──→ 已加载
       │
       └──(失败)──→ 错误状态 ──(重试)──→ 初始加载中
```

## 5. 跨页面参数

| 关联页面 | 关联方式 | 传出参数 | 接收参数 |
|---------|---------|---------|---------|
| 爆款视频库 | 返回链接（router.back） | 无 | 无（浏览器后退保留筛选状态） |
| 选题工作台 | "用作对标"按钮跳转 (router.push) | video_ids[]: number[], video_titles[]: string[] | 选题工作台通过路由参数 video_ids[] 接收对标视频ID数组 |
| 对标账号管理 | 来源账号点击跳转 (router.push) | account_id: number | 对标账号管理通过路由参数 $route.query.account_id 接收 |

### 路由定义

- 页面路径：`/videos/detail`
- 必需参数：`video_id`（query参数）
- 来源页面回退：利用浏览器 history，保留爆款视频库筛选状态
- 选题跳转：`/workspace?video_ids[]={id}&video_titles[]={title}`（数组格式，即使单个视频也用数组）

## 6. 待确认项

| # | 待确认内容 | 影响范围 | 建议方案 |
|---|-----------|---------|---------|
| 1 | 转发数(share_count)是否在爆款视频表中存储 | 互动数据面板 | 若数据库无此字段需新增；或考虑作为扩展字段 |
| 2 | 文案提取是同步还是异步 | 重新提取交互 | 建议异步：提交后轮询状态（3秒间隔），避免长时间阻塞 |
| 3 | AI拆解是同步还是异步 | AI拆解交互 | 建议异步：提交后轮询（3秒间隔），前端显示骨架屏3-8秒 |
| 4 | 文案编辑是否需要版本历史 | 保存编辑功能 | 初期建议只保存最新版本，不做版本管理 |
| 5 | 编辑文案后未保存离开的确认提示实现方式 | 路由守卫 | 使用 Vue Router 的 beforeRouteLeave 守卫 + window.beforeunload |
| 6 | 下载视频的来源：是代理下载还是直链 | 下载视频功能 | 建议后端代理下载（避免暴露源站URL），大文件考虑流式传输 |
| 7 | 视频详情中是否展示互动率 | 数据展示 | 页面描述中未提及，建议不加（详情页数据项已较多） |
