### 数据类型定义（TypeScript interface）

```typescript
// 平台筛选枚举
type PlatformFilter = 'all' | 'xiaohongshu' | 'douyin' | 'weixin'

// 监控状态枚举
type MonitorStatus = 'active' | 'paused' | 'collecting' | 'error'

// 对标账号
interface BenchmarkAccount {
  id: string                       // 📥存储 benchmark_accounts.id
  account_name: string             // 📥存储 benchmark_accounts.account_name
  platform: string                 // 📥存储 benchmark_accounts.platform
  platform_uid: string | null      // 📥存储 benchmark_accounts.platform_uid
  avatar_url: string | null        // 📥存储 benchmark_accounts.avatar_url
  homepage_url: string             // 📥存储 benchmark_accounts.homepage_url
  follower_count: number           // 📥存储 benchmark_accounts.follower_count
  content_direction: string        // 📥存储 benchmark_accounts.content_direction
  monitor_status: MonitorStatus    // 📥存储 benchmark_accounts.monitor_status
  last_collected_at: string | null // 📥存储 benchmark_accounts.last_collected_at
}

// 监控状态文字映射：📊计算
// active → "监控中", paused → "已暂停", collecting → "采集中", error → "采集失败"
// 监控开关状态：📊计算 monitor_status==='active' → true

// 账号统计摘要
interface AccountStats {
  total: number                    // 📊计算 COUNT(benchmark_accounts)
  monitoring: number               // 📊计算 COUNT WHERE monitor_status='active'
  paused: number                   // 📊计算 COUNT WHERE monitor_status='paused'
}

// 账号列表API响应
interface AccountListResponse {
  items: BenchmarkAccount[]
  total: number                    // 📊计算 总数
}

// 账号统计API响应
interface AccountStatsResponse {
  total: number                    // 📊计算
  monitoring: number               // 📊计算
  paused: number                   // 📊计算
}

// 添加/编辑账号请求
interface AccountFormRequest {
  account_name: string             // 📥存储 表单输入
  platform: string                 // 📥存储 表单下拉
  homepage_url: string             // 📥存储 表单输入
  follower_count?: number          // 📥存储 表单数字输入，默认0
  content_direction: string        // 📥存储 表单输入
}

// 切换监控状态请求
interface MonitorToggleRequest {
  monitor_status: 'active' | 'paused'  // 📥存储
}

// 创建采集任务请求（立即采集）
interface CollectTaskRequest {
  target_account_id: string        // 🔗调用 benchmark_accounts.id
  task_type: 'video_list'          // 📥存储
  max_count: number                // 📥存储 默认100
}

// 删除响应
interface DeleteResponse {
  success: boolean
}
```

| `toggleMonitor` | PATCH | `/api/v1/benchmark-accounts/:id/monitor` | `MonitorToggleRequest` | `{ id: string, monitor_status: MonitorStatus }` | 切换监控状态（开启/暂停） |

## 5. 状态管理

| 数据 | 归属（组件ref/Pinia/URL） | 说明 |
|------|--------------------------|------|
| 账号列表 (accounts[]) | Pinia: `benchmarkAccountStore` | 页面挂载时 GET /benchmark-accounts；平台筛选切换、搜索(防抖500ms)、增删改后刷新 |
| 平台筛选值 (platform_filter) | URL参数 + 组件ref | 默认 "all"；用户点击筛选标签切换 |
| 搜索关键词 (search_keyword) | 组件ref | 默认 ""；防抖500ms后触发列表刷新 |
| 统计摘要 (stats) | Pinia: `benchmarkAccountStore` | 页面挂载时 GET /benchmark-accounts/stats；添加/删除/监控切换后刷新 |
| 添加/编辑弹窗可见 (formModalVisible) | 组件ref | 默认 false；点击"添加账号"/"编辑"按钮切换 |
| 编辑模式 (editingAccount) | 组件ref | null(添加) / account对象(编辑)；点击"编辑"按钮设置 |
| 删除确认弹窗可见 (deleteModalVisible) | 组件ref | 默认 false；点击"删除"按钮切换 |
| 待删除账号 (deletingAccount) | 组件ref | null；点击"删除"按钮时设置 |
| 加载状态 (loading) | 组件ref | 默认 false；API请求中 |

### 加载策略

1. **页面挂载**：并行发起 2 个请求（accounts列表, stats统计），独立 loading
2. **骨架屏**：8张卡片骨架（4x2网格），220px高度，pulse 1.5s
3. **搜索防抖**：输入后 500ms 延迟发起请求
4. **操作后刷新**：添加/编辑/删除/监控切换后，同时刷新列表和统计摘要
5. **卡片局部更新**：监控开关切换、立即采集仅更新对应卡片状态，不刷新整个列表
