## 2. 组件 Props 定义

### 2.1 TypeScript 数据类型

```typescript
/** 平台枚举 */
type Platform = 'xiaohongshu' | 'douyin' | 'weixin';

/** 文案提取状态枚举 */
type TranscriptStatus = 'extracted' | 'pending' | 'failed' | 'extracting';

/** 排序字段枚举 */
type SortField = 'interaction_rate' | 'play_count' | 'published_at' | 'collect_count';

/** 排序方向枚举 */
type SortOrder = 'asc' | 'desc';

/** 视图模式枚举 */
type ViewMode = 'grid' | 'list';

/** 时间范围枚举 */
type TimeRange = '7d' | '30d' | 'all';

/** 爆款视频列表项 */
interface ViralVideoItem {
  id: number;                          // 📥存储 爆款视频.id
  account_id: number;                  // 📥存储 爆款视频.account_id
  account_name: string;                // 📥存储 对标账号.account_name（JOIN）
  platform: Platform;                  // 📥存储 爆款视频.platform
  thumbnail_url: string;               // 📥存储 爆款视频.thumbnail_url
  video_duration: string;              // 📥存储 爆款视频.video_duration，格式 MM:SS
  title: string;                       // 📥存储 爆款视频.title
  play_count: number;                  // 📥存储 爆款视频.play_count
  like_count: number;                  // 📥存储 爆款视频.like_count
  comment_count: number;               // 📥存储 爆款视频.comment_count
  collect_count: number;               // 📥存储 爆款视频.collect_count
  interaction_rate: number;            // 📊计算 (点赞+评论+收藏)/播放量*100，播放量为0时返回0
  transcript_status: TranscriptStatus; // 📥存储 爆款视频.transcript_status
  published_at: string;                // 📥存储 爆款视频.published_at，格式 ISO 8601
}

/** 爆款视频列表API响应 */
interface ViralVideoListResponse {
  total: number;                       // 📊计算 用于显示视频总数
  page: number;
  page_size: number;
  items: ViralVideoItem[];
}

/** 对标账号简要信息（筛选用） */
interface BenchmarkAccountBrief {
  id: number;                          // 📥存储 对标账号.id
  account_name: string;                // 📥存储 对标账号.account_name
  platform: Platform;                  // 📥存储 对标账号.platform
}

/** 对标账号列表API响应 */
interface BenchmarkAccountListResponse {
  items: BenchmarkAccountBrief[];
}

/** 筛选条件 */
interface FilterState {
  platform: string;                    // 📥存储 'all' | Platform
  account_id: number | null;           // 📥存储 对标账号.id
  time_range: TimeRange;               // 📥存储 时间范围
  min_engagement: number | null;       // 📥存储 互动量阈值
  keyword: string;                     // 📥存储 搜索关键词
}

/** 排序条件 */
interface SortState {
  sort_by: SortField;                  // 📥存储 排序字段
  order: SortOrder;                    // 📥存储 排序方向
}

/** 批量分析请求 */
interface AnalyzeBatchRequest {
  video_ids: number[];                 // 1-5个
}

/** 批量分析响应 */
interface AnalyzeBatchResponse {
  success: boolean;
  task_id: number;
}

/** 导出请求 */
interface ExportRequest {
  video_ids: number[];
  format: string;                      // 默认 'xlsx'
}
```

## 3. API 服务函数

### 3.1 查询类

```typescript
/**
 * 获取爆款视频列表
 * GET /api/v1/viral-videos
 */
function getViralVideos(params: {
  page?: number;                       // 默认 1
  page_size?: number;                  // 默认 12
  platform?: Platform;                 // 可选：xiaohongshu/douyin/weixin
  account_id?: number;                 // 可选：对标账号ID
  time_range?: TimeRange;              // 可选：7d/30d/all
  min_engagement?: number;             // 可选：互动量阈值
  sort_by?: SortField;                 // 默认 interaction_rate
  order?: SortOrder;                   // 默认 desc
  keyword?: string;                    // 可选：按标题模糊搜索
}): Promise<ViralVideoListResponse>

/**
 * 获取对标账号简要列表（筛选下拉用）
 * GET /api/v1/benchmark-accounts?brief=true
 */
function getBenchmarkAccountsBrief(): Promise<BenchmarkAccountListResponse>
```

### 3.2 操作类


## 4. 数据库表结构

### 4.1 爆款视频表 (viral_videos)

| 字段名 | 类型 | 约束 | 说明 | 溯源 |
|--------|------|------|------|------|
| id | SERIAL | PRIMARY KEY | 主键 | 📥存储 |
| account_id | INTEGER | NOT NULL, FOREIGN KEY → benchmark_accounts(id), INDEX | 对标账号ID（外键索引） | 📥存储 |
| platform | VARCHAR(20) | NOT NULL, INDEX | 平台：xiaohongshu/douyin/weixin | 📥存储 |
| title | VARCHAR(500) | NOT NULL | 视频标题 | 📥存储 |
| thumbnail_url | VARCHAR(1000) | | 缩略图URL | 📥存储 |
| video_url | VARCHAR(1000) | | 视频播放地址 | 📥存储 |
| video_duration | VARCHAR(10) | | 视频时长，格式 MM:SS | 📥存储 |
| play_count | INTEGER | DEFAULT 0 | 播放量 | 📥存储 |
| like_count | INTEGER | DEFAULT 0 | 点赞数 | 📥存储 |
| comment_count | INTEGER | DEFAULT 0 | 评论数 | 📥存储 |
| collect_count | INTEGER | DEFAULT 0 | 收藏数 | 📥存储 |
| interaction_rate | DECIMAL(5,2) | DEFAULT 0, INDEX | 互动率（百分比） | 📊计算 |
| transcript_status | VARCHAR(20) | DEFAULT 'pending' | 文案状态：pending/extracting/extracted/failed | 📥存储 |
| published_at | TIMESTAMP WITH TIME ZONE | INDEX | 发布时间 | 📥存储 |
| created_at | TIMESTAMP WITH TIME ZONE | DEFAULT NOW() | 创建时间 | 📥存储 |
| updated_at | TIMESTAMP WITH TIME ZONE | DEFAULT NOW() | 更新时间 | 📥存储 |

**索引清单**：
- `idx_viral_videos_account_id` ON (account_id) — 外键索引，筛选+JOIN
- `idx_viral_videos_platform` ON (platform) — 平台筛选
- `idx_viral_videos_published_at` ON (published_at) — 时间排序
- `idx_viral_videos_interaction_rate` ON (interaction_rate) — 互动率排序

### 4.2 对标账号表 (benchmark_accounts) — 参考引用

| 字段名 | 类型 | 约束 | 说明 | 溯源 |
|--------|------|------|------|------|
| id | SERIAL | PRIMARY KEY | 主键 | 📥存储 |
| account_name | VARCHAR(100) | NOT NULL | 账号名称 | 📥存储 |
| platform | VARCHAR(20) | NOT NULL | 平台 | 📥存储 |

## 5. 状态管理

### 5.1 数据归属

| 数据 | 存储位置 | 类型 | 说明 |
|------|---------|------|------|
| 视频列表 (videos) | 组件状态 ref | `ViralVideoItem[]` | 页面加载/筛选/搜索/翻页时更新 |
| 分页信息 (page, page_size, total) | 组件状态 ref | `{ page: number, page_size: number, total: number }` | 分页加载 |
| 筛选条件 (filters) | 组件状态 reactive | `FilterState` | 默认：all/null/7d/null/'' |
| 排序条件 (sortState) | 组件状态 ref | `SortState` | 默认：interaction_rate/desc |
| 搜索关键词 (keyword) | 组件状态 ref | `string` | 防抖500ms后触发筛选 |
| 视图模式 (viewMode) | 本地存储 localStorage | `ViewMode` | 默认 grid，跨会话保留 |
| 已选视频ID列表 (selected_ids) | 组件状态 ref | `number[]` | 默认 []，勾选/全选/取消时更新 |
| 加载状态 (loading) | 组件状态 ref | `boolean` | 默认 true |
| 加载更多状态 (loading_more) | 组件状态 ref | `boolean` | 默认 false |
| 对标账号选项列表 (account_options) | 组件状态 ref | `BenchmarkAccountBrief[]` | 页面加载时获取一次 |

### 5.2 计算属性

| 属性名 | 类型 | 计算逻辑 | 说明 |
|--------|------|---------|------|
| all_loaded | `boolean` | `videos.length >= total` | 📊计算 是否全部加载完 |
| selected_count | `number` | `selected_ids.length` | 📊计算 已选计数 |
| video_total | `number` | `total`（来自API响应） | 📊计算 视频总数 |

### 5.3 加载策略

| 时机 | 触发 | API调用 | 备注 |
|------|------|---------|------|
| 页面初始加载 | `onMounted` | `getViralVideos()` + `getBenchmarkAccountsBrief()` | 并行请求 |
| 筛选变更 | watch filters | `getViralVideos()` | 重置 page=1 |
| 排序变更 | watch sortState | `getViralVideos()` | 重置 page=1 |
| 搜索输入 | watch keyword (debounce 500ms) | `getViralVideos()` | 重置 page=1 |
| 加载更多 | 滚动到底部 | `getViralVideos(page+1)` | 追加到 videos 数组 |
| 加入分析 | 点击按钮 | `analyzeBatch()` | 成功后跳转选题工作台 |
| 导出 | 点击按钮 | `exportViralVideos()` | 触发浏览器下载 |
