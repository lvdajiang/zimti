```typescript
/** 平台枚举 */
type Platform = 'xiaohongshu' | 'douyin' | 'weixin';

/** 文案提取状态枚举 */
type TranscriptStatus = 'extracted' | 'pending' | 'failed' | 'extracting';

/** 文案信息 */
interface TranscriptInfo {
  id: number;                          // 📥存储 文案提取.id
  transcript_text: string | null;      // 📥存储 文案提取.transcript_text
  extraction_method: string | null;    // 📥存储 文案提取.extraction_method
  extracted_at: string | null;         // 📥存储 文案提取.extracted_at
  status: TranscriptStatus;            // 📥存储 文案提取.status
}

/** 爆款拆解信息 */
interface AnalysisInfo {
  id: number;                          // 📥存储 爆款拆解.id
  hook_type: string[];                 // 📥存储 爆款拆解.hook_type，标签组
  content_structure: string;           // 📥存储 爆款拆解.content_structure
  conversion_method: string[];         // 📥存储 爆款拆解.conversion_method，标签组
  highlight_tags: string[];            // 📥存储 爆款拆解.highlight_tags，标签组
  analyzed_at: string;                 // 📥存储 爆款拆解.analyzed_at
}

/** 视频详情 */
interface VideoDetail {
  id: number;                          // 📥存储 爆款视频.id
  account_id: number;                  // 📥存储 爆款视频.account_id
  account_name: string;                // 📥存储 对标账号.account_name（JOIN）
  platform: Platform;                  // 📥存储 爆款视频.platform
  video_url: string;                   // 📥存储 爆款视频.video_url
  thumbnail_url: string;               // 📥存储 爆款视频.thumbnail_url

  like_count: number;                  // 📥存储 爆款视频.like_count
  comment_count: number;               // 📥存储 爆款视频.comment_count

  share_count: number;                 // 📥存储 爆款视频.share_count

interface ExtractTranscriptResponse {
```

## 3. API 服务函数

### 3.1 查询类

```typescript
/**
 * 获取视频详情（含文案和拆解数据）
 * GET /api/v1/viral-videos/:id
 */
function getViralVideoDetail(id: number): Promise<VideoDetail>
```

### 3.2 操作类

```typescript
/**
 * 重新提取文案（异步）
 * POST /api/v1/viral-videos/:id/transcript/extract
 */
function extractTranscript(id: number): Promise<ExtractTranscriptResponse>

/**
 * AI拆解（异步）
 * POST /api/v1/viral-videos/:id/analyze
 */
function analyzeVideo(id: number): Promise<AnalyzeVideoResponse>

/**
 * 保存文案编辑
 * PUT /api/v1/viral-videos/:id/transcript
 */
function saveTranscript(id: number, data: SaveTranscriptRequest): Promise<SaveTranscriptResponse>

/**
 * 用作对标（加入选题）
 * POST /api/v1/topic-proposals/add-videos
 */
function addToTopic(data: AddToTopicRequest): Promise<AddToTopicResponse>

/**
 * 下载视频
 * GET /api/v1/viral-videos/:id/download
 * 返回二进制文件流，触发浏览器下载
 */
function downloadVideo(id: number): Promise<Blob>
```

### 3.3 轮询辅助

```typescript
/**
 * 轮询视频详情（文案提取/AI拆解后使用）
 * 复用 getViralVideoDetail，间隔 3 秒
 * 停止条件：transcript_status 不再是 extracting / analysis 不再为 null
 */
function pollVideoDetail(id: number, interval?: number): Promise<VideoDetail>
```

## 5. 状态管理

### 5.1 数据归属

| 数据 | 存储位置 | 类型 | 说明 |
|------|---------|------|------|
| 视频详情 (video_detail) | 组件状态 ref | `VideoDetail \| null` | 页面加载/操作完成后更新 |
| 文案提取状态 (transcript_status) | 组件状态 computed | `TranscriptStatus` | 从 video_detail.transcript_status 计算 |
| AI拆解状态 (analysis_status) | 组件状态 computed | `'none' \| 'analyzing' \| 'done' \| 'failed'` | 从 video_detail.analysis + analyzing 计算 |
| 文案编辑模式 (is_editing) | 组件状态 ref | `boolean` | 默认 false，编辑/取消按钮切换 |
| 编辑中文案内容 (editing_text) | 组件状态 ref | `string` | 进入编辑时复制 transcript_text |
| 拆解加载中 (analyzing) | 组件状态 ref | `boolean` | 默认 false，AI拆解按钮点击/完成 |
| 提取加载中 (extracting) | 组件状态 ref | `boolean` | 默认 false，重新提取按钮点击/完成 |
| 保存加载中 (saving) | 组件状态 ref | `boolean` | 默认 false，保存按钮点击/完成 |
| 页面加载状态 (loading) | 组件状态 ref | `boolean` | 默认 true |
| 页面错误状态 (error) | 组件状态 ref | `string \| null` | 默认 null |
| 轮询定时器 (poll_timer) | 组件状态 ref | `ReturnType\<typeof setInterval\> \| null` | 提取/拆解后3秒轮询 |
| 视频 ID (video_id) | URL 参数 | `number` | 从 $route.query.video_id 获取 |

### 5.2 计算属性

| 属性名 | 类型 | 计算逻辑 | 说明 |
|--------|------|---------|------|
| has_unsaved_changes | `boolean` | `is_editing && editing_text !== video_detail?.transcript?.transcript_text` | 📊计算 未保存提示 |
| transcript_text | `string \| null` | `video_detail?.transcript?.transcript_text` | 📥存储 文案全文 |
| is_analyzed | `boolean` | `video_detail?.analysis?.analyzed_at != null` | 📊计算 是否已拆解 |

### 5.3 加载策略

| 时机 | 触发 | API调用 | 备注 |
|------|------|---------|------|
| 页面初始加载 | `onMounted` | `getViralVideoDetail(video_id)` | 全页面数据一次加载 |
| 重新提取文案 | 按钮点击 | `extractTranscript(video_id)` → 轮询 `getViralVideoDetail` | 异步，3秒间隔轮询 |
| AI拆解 | 按钮点击 | `analyzeVideo(video_id)` → 轮询 `getViralVideoDetail` | 异步，3秒间隔轮询 |
| 保存文案编辑 | 保存按钮 | `saveTranscript(video_id, data)` | 成功后刷新详情 |
| 用作对标 | 按钮点击 | `addToTopic(data)` | 成功后跳转选题工作台 |
| 下载视频 | 控制栏下载 | `downloadVideo(video_id)` | 触发浏览器下载 |
| 复制文案 | 按钮点击 | 无API，`navigator.clipboard.writeText` | 纯前端操作 |

### 5.4 路由守卫

- **离开守卫** (`beforeRouteLeave`)：当 `has_unsaved_changes` 为 true 时，弹出确认对话框"文案修改未保存，确定离开？"
- **页面卸载** (`window.beforeunload`)：同样检测 `has_unsaved_changes`

## 6. 路由配置

```typescript
const routes: RouteRecordRaw[] = [
```

### 6.1 跨页面跳转

| 目标页面 | 跳转方式 | 传出参数 | 路由格式 |
|---------|---------|---------|---------|

| 爆款视频库 | video_id（query，必填） | 展示指定视频详情 |
