## 八、版权合规要求

| 规则 | 说明 |
|------|------|
| 优先使用CC0/Pexels License素材 | 无需署名，最省心 |
| CC-BY素材必须标注来源 | 系统自动在视频结尾添加"素材来源"页 |
| 平台AIGC标识 | AI生成素材必须标注，发布工作台会提醒 |
| 定期审查版权状态 | 素材库的copyright_status字段需定期更新 |
| 国内平台特别注意 | 抖音/小红书对音乐版权审核严格，优先使用Artlist等正版平台 |

> 参见需求文档「合规底线」：素材只用免费可商用来源或AI生成，不碰侵权素材。
