## 3. 字体规范

### 字体家族

- 全局字体: `"Inter", "SF Pro Display", -apple-system, BlinkMacSystemFont, "Segoe UI", sans-serif`
- 数字字体: `"DIN Alternate", "Inter", sans-serif`（数据指标、百分比、播放量等）

### 字号层级

| 层级 | 变量 | 应用场景 |
|------|------|---------|
| 页面标题 | $text-3xl (24px) | "任务中心" |
| 数据指标数值 | $text-2xl (20px) | 昨日播放量、爆款视频数等核心数字 |
| 区块标题 | $text-md (14px) | "今日任务"、"数据速览"、"热点推荐"等区块标题 |
| 空状态标题 | $text-xl (18px) | "创建你的第一个任务" |
| 卡片标题 | $text-lg (16px) | 快捷操作模块名称 |
| 进度百分比 | $text-lg (16px) | 任务进度数字 |
| 正文/任务标题 | $text-md (14px) | 任务卡片标题、快捷操作描述 |
| 日期显示 | $text-md (14px) | 当前日期 |
| 辅助文字 | $text-sm (12px) | 工作流步骤文字、指标名称、趋势描述 |
| 标签文字 | $text-xs (11px) | 任务状态标签、来源平台、匹配度评分 |

### 字重应用

| 字重变量 | 应用 |
|----------|------|
| $weight-bold (700) | 页面标题、数据指标数值 |
| $weight-semibold (600) | 任务标题、区块标题、工作流当前步骤、进度百分比、快捷操作模块名称 |
| $weight-medium (500) | 任务筛选标签选中态、按钮文字、热点标题、工作流已完成步骤 |
| $weight-normal (400) | 辅助说明文字、标签文字、日期显示、趋势描述 |

### 行高

- 标题行高: 1.3
- 正文行高: 1.5
- 标签行高: 1.4
- 数据数值行高: 1.2
