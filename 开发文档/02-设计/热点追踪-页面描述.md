# 页面描述：热点追踪

> 生成日期：2026-05-09
> 前置文档：热点追踪-设计规格.md / 自媒体内容生产系统-需求.md
> 页面类型：桌面后台列表+筛选页
> 尺寸：1920x1080

---

## 1. 页面总览

- **页面目标**：运营者追踪各平台热点话题，查看相关度评分和AI结合建议，一键将热点带入选题工作台
- **角色权限**：运营者
- **进入方式**：Sidebar"热点追踪"导航项；仪表盘快捷操作"热点追踪"入口；仪表盘热点推荐"查看更多"

## 6. 边缘场景

| 场景 | 处理方式 |
|------|---------|
| 相关度<30%的热点 | 卡片 opacity 0.6 灰显但仍展示 |
| 超过48小时未使用 | 自动标记"已过期"，卡片 opacity 0.5 |
| 24小时有效期 | 左侧 3px $status-error 竖条 + "紧急"标签 |
| 敏感话题 | 自动过滤，不展示 |
| 关键词匹配推送 | 弹窗提醒 + 列表高亮 |
| 无监控关键词 | 趋势曲线区域隐藏，仅显示"+ 添加"按钮 |
| 热度值极大（>100万） | 格式化为 "100万+" |
| 手动录入重复热点 | 提示"该热点已存在" |
| 快速连续刷新 | 防抖，避免重复请求 |
