## 7. 响应式规则

### 断点定义

| 断点 | 宽度范围 | 适用设备 |
|------|---------|---------|
| xl | >= 1280px | 标准桌面 |
| lg | 1024px ~ 1279px | 小桌面 |
| md | 768px ~ 1023px | 平板横屏 |
| sm | < 768px | 平板竖屏 / 移动端 |

### xl（>= 1280px，标准桌面）

| 属性 | 值 |
|------|-----|
| 表格列 | 全部展示（10列） |
| Content 内边距 | $space-3xl (32px) |
| 搜索框宽度 | 240px |

### lg（1024px ~ 1279px）

| 属性 | 值 |
|------|-----|
| 表格列 | 隐藏"创建时间"列（9列） |
| Content 内边距 | $space-2xl (24px) |
| 搜索框宽度 | 200px |

### md（768px ~ 1023px）

| 属性 | 值 |
|------|-----|
| 表格列 | 隐藏"创建时间""耗时"列（8列） |
| Content 内边距 | $space-xl (20px) |
| 搜索框宽度 | 180px |
| 统计摘要 | 移至标题下方独立一行 |

### sm（< 768px）

| 属性 | 值 |
|------|-----|
| 表格 | 改为卡片列表布局（每卡片一个任务） |
| Content 内边距 | $space-lg (16px) |

| 搜索框 | 全宽 |
| 统计摘要 | 网格 2x2 布局 |
