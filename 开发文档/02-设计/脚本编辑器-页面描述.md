# 页面描述：脚本编辑器

> 生成日期：2026-05-09
> 前置文档：脚本编辑器-设计规格.md / 自媒体内容生产系统-需求.md
> 页面类型：桌面后台编辑器页
> 尺寸：1920x1080

---

## 1. 页面总览

- **页面目标**：运营者将口播文案升级为包含导演思维的分镜脚本，编辑视觉/听觉/增强指令，进行去AI味检查后跳转视频生成
- **角色权限**：运营者
- **进入方式**：选题工作台"编辑脚本"按钮跳转（带入选题和骨架数据）

## 6. 边缘场景

| 场景 | 处理方式 |
|------|---------|
| 无文案就点"生成导演脚本" | 按钮禁用，tooltip 提示"请先输入口播文案" |
| 脚本不完整就点"生成视频" | 按钮灰色禁用态，tooltip 提示"脚本不完整或未通过去AI味检查" |
| 去AI味检查有严重问题 | "生成视频"按钮禁用，必须修改禁止词后才能继续 |
| 超长文案（>5000字） | 文案区支持滚动，分镜段落数量相应增多 |
| 快速连续点击"生成导演脚本" | 按钮禁用+防抖，上次请求未完成时不可再次触发 |
| 拖拽到自身位置 | 不触发任何操作 |
| 删除唯一段落 | 弹出确认"确定删除最后一个段落？" |
| 编辑后未保存 | 顶部黄色提示条"有未保存的修改"+ 保存/放弃按钮 |
| 口播占比根据视频类型自动调整阈值 | 知识干货型≥70%，情感故事型50-70%，爆款引流型30-50% |
| 口播音频生成中再次点击 | 按钮禁用，等待上次完成 |
| 返回选题工作台时有未保存修改 | 弹出确认"有未保存的修改，确定离开？" |
