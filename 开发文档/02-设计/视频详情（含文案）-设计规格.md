## 3. 字体规范

### 字体家族

- 全局字体: `"Inter", "SF Pro Display", -apple-system, BlinkMacSystemFont, "Segoe UI", sans-serif`
- 数字字体: `"DIN Alternate", "Inter", sans-serif`
- 文案字体: 等宽字体可选（编辑态）`"JetBrains Mono", "Fira Code", monospace`

### 字号层级

| 层级 | 变量 | 应用场景 |
|------|------|---------|
| 视频标题 | $text-xl (18px) | 视频标题 |
| 区块标题 | $text-md (14px) | "口播文案""爆款拆解" |
| 拆解子标题 | $text-lg (16px) | "钩子类型""内容结构" |
| 互动数据数值 | $text-lg (16px) | 播放量、点赞等 |
| 正文/文案 | $text-md (14px) | 文案正文、拆解内容 |
| 返回链接 | $text-md (14px) | "← 返回爆款视频库" |
| 辅助文字 | $text-sm (12px) | 发布时间、互动数据标签 |
| 标签文字 | $text-xs (11px) | 爆款拆解标签、亮点标签 |
| 按钮文字 | $text-sm (12px) | 操作按钮 |

### 字重应用

| 字重变量 | 应用 |
|----------|------|
| $weight-bold (700) | 互动数据数值 |
| $weight-semibold (600) | 视频标题、区块标题、拆解子标题、来源账号 |
| $weight-medium (500) | 返回链接、按钮文字、拆解标签、亮点标签 |
| $weight-normal (400) | 文案正文、辅助说明、发布时间 |

### 行高

- 标题行高: 1.3
- 正文行高: 1.6（文案区可读性优先）
- 标签行高: 1.4
- 数据数值行高: 1.2

## 6. 状态设计

### 6.1 文案提取状态

| 状态 | 文案区表现 | 操作按钮 |
|------|-----------|---------|
| 已提取 | 显示完整文案文本 | 编辑/复制可用 |
| 待提取 | 空状态 + "提取文案"按钮 | 编辑/复制禁用 |
| 提取中 | 骨架屏 + "文案提取中..." | 所有文案操作禁用 |
| 提取失败 | 错误提示 + "重新提取"按钮 | 重新提取可用 |

### 6.2 爆款拆解状态

| 状态 | 拆解区表现 |
|------|-----------|
| 未拆解 | "点击 AI拆解 分析该视频的爆款结构" 引导卡片 |
| 分析中 | 骨架屏 + "AI 正在分析..." |
| 已拆解 | 完整拆解内容展示 |
| 分析失败 | 错误提示 + "重试"按钮 |

### 6.3 交互状态

**视频播放器**

| 状态 | 表现 |
|------|------|
| 默认 | 显示视频首帧 + 居中播放按钮 |
| 悬停 | 显示控制栏 |
| 播放中 | 控制栏自动隐藏，3秒无操作后隐藏 |
| 暂停 | 显示控制栏 + 居中播放按钮 |

**文案文本区**

| 状态 | 边框 | 背景 |
|------|------|------|
| 只读 | 无 | $bg-primary |
| 编辑态-聚焦 | 1px $brand-indigo | $bg-primary |
| 编辑态-失焦 | 1px $border-light | $bg-primary |

**操作按钮**

| 状态 | 表现 |
|------|------|
| 默认 | 正常样式 |
| 悬停 | 对应悬停色 |
| 加载中 | spinner + 禁用态 |
| 禁用 | $text-disabled, cursor: not-allowed |

### 6.4 加载状态

| 状态 | 表现 |
|------|------|
| 首次加载 | 左栏视频播放器骨架 + 右栏信息骨架 + 文案区骨架 |
| 网络错误 | 红色提示条 + "加载失败，点击重试" |

## 7. 响应式规则

### 断点定义

| 断点 | 宽度范围 | 适用设备 |
|------|---------|---------|
| xl | >= 1280px | 标准桌面 |
| lg | 1024px ~ 1279px | 小桌面 |
| md | 768px ~ 1023px | 平板横屏 |
| sm | < 768px | 平板竖屏 / 移动端 |

### xl（>= 1280px，标准桌面）

| 属性 | 值 |
|------|-----|

| Content 内边距 | $space-3xl (32px) |
| 主内容布局 | 双栏（左58% 右42%） |

### lg（1024px ~ 1279px）

| 属性 | 值 |
|------|-----|
| 内容容器宽度 | 100% |
| Content 内边距 | $space-2xl (24px) |
| 主内容布局 | 双栏（左55% 右45%） |

### md（768px ~ 1023px）

| 属性 | 值 |
|------|-----|
| 内容容器宽度 | 100% |
| Content 内边距 | $space-xl (20px) |
| 主内容布局 | 单栏（视频预览在上，信息+文案在下） |
