> 生成日期：2026-05-09

> 尺寸：1920x1080

---

## 1. 页面总览

- **页面目标**：运营者基于爆款数据分析和热点信息，生成当日选题方案和脚本骨架，选择或融合选题后进入脚本编辑器
- **角色权限**：运营者
- **进入方式**：Sidebar"选题工作台"导航项；仪表盘快捷操作"选题工作台"入口；热点追踪"用于选题"按钮跳转；爆款视频库"加入分析"按钮跳转

## 6. 边缘场景

| 场景 | 处理方式 |
|------|---------|
| 未选择任何素材就点"换一批" | 提示"请先选择至少1个对标视频或热点" |
| 选择超过5个对标视频 | 第6个点击无效，提示"最多选择5个对标视频" |
| 融合选题时只选了1个 | "融合选题"按钮禁用，tooltip 提示"需选择≥2个选题" |
| 融合后用户不满意 | 可点击"换一批"重新生成，融合结果被清除 |
| AI生成超时（>30秒） | 显示"生成时间较长，请稍候..."，超时后显示"生成超时，请重试" |
| 快速连续点击"换一批" | 按钮禁用+防抖，上次请求未完成时不可再次触发 |
| 口播占比根据选题类型自动调整 | 知识干货型≥70%，情感故事型50-70%，爆款引流型30-50% |

| 编辑脚本按钮未选定时 | 灰色禁用态，不可点击 |
