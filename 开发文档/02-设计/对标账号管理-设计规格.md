## 1. 布局结构

### 整体布局

```
┌─────────────────────────────────────────────────────────┐

│  │ "对标账号管理" + 账号统计摘要 + [添加账号] 按钮        │ │
```

## 7. 响应式规则

### 断点定义

| 断点 | 宽度范围 | 适用设备 |

| md | 768px ~ 1023px | 平板横屏 |
| sm | < 768px | 平板竖屏 / 移动端 |

| 账号名称 | 卡片 | 头像右侧 | $text-lg, $weight-semibold, $text-primary |
| 平台 | 卡片 | 名称下方平台标签 | $text-xs, $weight-medium, 平台专属色 |
| 粉丝数 | 卡片 | 数据区第一项 | 标签 $text-xs $text-tertiary, 数值 $text-md $weight-medium $text-primary |
| 内容方向 | 卡片 | 平台标签下方 | $text-xs, $weight-normal, $bg-secondary, $text-secondary |
| 监控状态 | 卡片 | 数据区与操作栏之间 | 状态点 + $text-xs $weight-medium, 颜色跟随状态 |
| 最近采集时间 | 卡片 | 底部 | $text-xs, $text-tertiary |

### 操作

| 操作 | 触发位置 | 交互方式 | 条件 |
|------|---------|---------|------|
| 添加账号 | 页面标题行右侧主按钮 | 弹窗表单 | 始终可用 |
| 编辑 | 卡片操作栏编辑图标 | 弹窗表单（预填数据） | 始终可用 |
| 删除 | 卡片操作栏删除图标 | 确认弹窗 | 始终可用 |
| 立即采集 | 卡片操作栏采集图标 | 触发采集 | 监控中可用，已暂停/采集中禁用 |
| 暂停监控 | 卡片内开关 | 切换开关状态 | 监控中时可用 |
| 恢复监控 | 卡片内开关 | 切换开关状态 | 已暂停时可用 |
