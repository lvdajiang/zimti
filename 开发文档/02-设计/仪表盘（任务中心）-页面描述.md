## 6. 边缘场景

| 场景 | 处理方式 |
|------|---------|
| 超长任务标题（>30字） | 截断显示+省略号，完整标题在悬停时 tooltip 显示 |
| 状态标签文字过长 | 状态文字固定映射，不会超长（待办/采集/选题/制作/发布/监控/已完成） |
| 进度为0% | 进度条不显示填充，百分比显示 "0%" |
| 进度为100% | 进度条满填充 $status-success，状态标签变为"已完成" |
| 数据指标为0 | 显示 "0"，趋势描述隐藏 |
| 无对标账号（爆款数为0） | 正常显示 "0"，趋势描述显示 "暂无对标数据" |
| 热点匹配度极低（<10%） | 匹配度进度条填充 $border-medium 灰色，评分数字正常显示 |
| 任务数量超过显示上限 | 列表底部显示提示条："已展示全部 {N} 个任务" |
| 无网络 | 复用错误状态，文案改为"网络连接失败，请检查网络后重试" |
| 数据过期 | 页面加载时自动拉取最新数据，无需手动刷新 |
| 新任务创建成功 | 列表顶部插入新卡片，带入场动画（300ms fadeIn + slideDown） |
| 任务状态自动推进 | 卡片状态标签平滑过渡（200ms），进度条动画填充 |
| 新热点推送 | 热点列表顶部插入新卡片，带闪烁提示（1s），3s后消失 |
| 数据指标更新 | 数值跳动动画（countUp 效果），趋势箭头同步更新 |
