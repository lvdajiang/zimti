## 1. 页面总览

- **页面目标**：运营者管理所有对标账号，按平台分类查看、添加、编辑、删除账号，控制监控开关
- **角色权限**：运营者
- **进入方式**：Sidebar"对标账号管理"导航项；仪表盘快捷操作入口

## 6. 边缘场景

| 场景 | 处理方式 |
|------|---------|
| 超长账号名称（>15字） | 截断显示+省略号，完整名称在悬停时 tooltip 显示 |
| 内容方向标签超长 | 截断显示+省略号，最大宽度 120px |
| 粉丝数为0 | 显示 "0" |
| 从未采集过 | 最近采集时间显示 "尚未采集" |
| 超3天未采集 | 最近采集时间文字变红色，右侧显示 "!" 警告图标 |
| 采集中状态 | 开关不可切换，立即采集按钮显示 loading，状态文字蓝色+旋转图标 |
| 采集失败状态 | 状态指示点红色，文字"采集失败"，隐藏立即采集按钮，显示重试采集按钮 |
| 删除最后一张卡片 | 网格显示空状态 |
| 表单验证失败 | 输入框边框变 $status-error，下方显示错误提示文字 |
| 主页链接格式错误 | 错误提示："请输入有效的账号主页链接" |
| 平台账号数超上限 | 无硬性上限，但建议3-5个，超出时不限制只展示 |
