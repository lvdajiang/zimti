## 6. 边缘场景

| 场景 | 处理方式 |
|------|---------|
| 超长目标描述（>40字） | 截断+省略号，悬停 tooltip 显示完整 |
| 采集超过100条 | 自动分批，进度条分阶段显示，提示条："单次采集超过100条，已自动分批" |
| 采集失败（平台反爬） | 状态标红，日志显示失败原因，可重试 |
| 采集失败自动重试1次后仍失败 | 标记为"待手动处理"，状态红色 |
| 文案提取无视频下载 | 日志提示"未找到可提取的视频"，状态失败 |
| 采集进行中关闭页面 | 后台继续执行，重新打开时恢复状态 |
| 全选删除 | 批量操作确认弹窗："确认删除选中的 N 个任务？" |
| 进度为0% | 进度条不显示填充，百分比显示 "0%" |
| 定时采集冲突 | 提示："当前时间已有定时任务，请调整时间" |
| 无对标账号可选 | 目标账号下拉为空，提示："请先添加对标账号" |
