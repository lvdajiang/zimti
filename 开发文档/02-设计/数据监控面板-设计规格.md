## 3. 字体规范

### 字体家族

- 全局字体: `"Inter", "SF Pro Display", -apple-system, BlinkMacSystemFont, "Segoe UI", sans-serif`
- 数字字体: `"DIN Alternate", "Inter", sans-serif`（概览数值、表格数字）
- 图表字体: `"Inter", sans-serif`

### 字号层级

| 层级 | 变量 | 应用场景 |
|------|------|---------|
| 页面标题 | $text-3xl (24px) | "数据监控" |
| 概览数值 | $text-2xl (20px) | 总播放量、完播率等 |
| 区块标题 | $text-md (14px) | AI分析报告、踩坑日志 |
| 正文 | $text-sm (12px) | 表格正文、AI分析内容、日志摘要 |
| 辅助文字 | $text-xs (11px) | 表头、概览标签、趋势对比、图例 |

### 字重应用

| 字重变量 | 应用 |
|----------|------|
| $weight-bold (700) | 页面标题、概览数值 |
| $weight-semibold (600) | AI结论标题、日志周数、表头 |
| $weight-medium (500) | 改进指令、趋势箭头、按钮文字 |
| $weight-normal (400) | 概览标签、表格正文、AI分析内容 |

### 行高

- 标题行高: 1.3
- 正文行高: 1.5
- 表格行高: 1.4
- 标签行高: 1.4

## 6. 状态设计

### 6.1 数据指标趋势

| 状态 | 颜色 | 表现 |
|------|------|------|
| 上升 | $status-success | ↑ + 百分比 |
| 下降 | $status-error | ↓ + 百分比 |
| 持平 | $text-tertiary | - "持平" |

### 6.2 AI分析状态

| 状态 | 指示 | 表现 |
|------|------|------|
| 未分析 | "请求数据分析" 按钮 | 默认可用 |
| 数据不足 | 禁用 + 提示 | "数据不足，至少需要3条记录" |
| 分析中 | 旋转loading | "AI 正在分析数据..." |
| 分析完成 | 绿色勾选 | 显示结论和改进建议 |
| 分析失败 | 红色提示 | "分析失败，点击重试" |

### 6.3 周记状态

| 状态 | 表现 |
|------|------|
| 本周未写 | "写本周总结"按钮高亮 + 周日晚提醒 |
| 本周已写 | 按钮变为"查看本周记录" |
| 历史记录 | 可展开查看完整三条 |

### 6.4 加载与空状态

| 状态 | 表现 |
|------|------|
| 首次加载 | 概览卡片骨架屏 + 表格骨架屏 |
| 无数据 | 空状态"暂无监控数据" + "发布视频后数据将自动展示" |
| 表格加载中 | 行骨架屏, 5行 |
| 趋势图加载中 | 图表区域骨架屏 |
| 网络错误 | 红色提示条 + "加载失败，点击重试" |
