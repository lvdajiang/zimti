> 生成日期：2026-05-09

## 1. 页面总览

- **页面目标**：运营者浏览和筛选对标账号的爆款视频，查看数据指标和文案状态，批量选择视频进入选题分析
- **角色权限**：运营者
- **进入方式**：Sidebar"爆款视频库"导航项；仪表盘快捷操作"爆款视频库"入口

## 6. 边缘场景

| 场景 | 处理方式 |
|------|---------|
| 超长视频标题（>40字） | 2行省略号，悬停 tooltip 显示完整 |
| 播放量/点赞等数据为0 | 显示 "0" |
| 播放量数值很大（>1万） | 格式化为 "1.2万" |
| 互动率极高（>50%） | 正常显示，不设上限 |
| 缩略图加载失败 | 显示灰色占位图+视频图标 |
| 文案提取中 | 角标显示 "提取中..." 蓝色+旋转图标 |
| 批量选择超过5个 | "加入分析"按钮禁用，提示"最多选择5个" |
| 无对标账号可选 | 对标账号下拉为空，提示"请先添加对标账号" |
| 全部视频已加载完 | 底部显示"已展示全部 N 条视频" |
| 网络断开 | 复用错误状态 |
