## 6. 状态设计

### 6.1 资产状态

| 状态 | 指示色 | 表现 |
|------|--------|------|
| 草稿 | $text-tertiary | 标签"草稿" |
| 已发布 | $status-success | 标签"已发布" |
| 已归档 | $text-tertiary | 标签"已归档", 卡片降低透明度 |

### 6.2 版权状态

| 状态 | 表现 |
|------|------|
| 有效 | 版权标签显示绿色 |
| 即将过期（<7天） | 版权标签显示黄色 + "即将过期" |
| 已过期 | 版权标签显示红色 + "已过期,请重新确认" |

### 6.3 AI自动标记

| 标签 | 触发条件 | 颜色 |
|------|---------|------|
| 高完播 | 完播率 >= 40% | $status-success |
| 高互动 | 互动率 >= 5% | $brand-indigo |
| 高转化 | 私信数 >= 10 | $status-warning |
| 爆款结构 | 播放量 >= 平均值 2倍 | $status-error |

### 6.4 交互状态

**资产卡片**

| 状态 | 边框 | 阴影 | 其他 |
|------|------|------|------|
| 默认 | 1px $border-light | 无 | - |
| 悬停 | 1px $border-medium | $hover-shadow | 操作按钮更明显 |
| 归档 | 1px $border-light | 无 | 透明度 0.7 |

**操作按钮**

| 状态 | 背景 | 图标颜色 |
|------|------|---------|
| 默认 | 透明 | $text-secondary |
| 悬停 | $bg-secondary | $text-primary |
| 禁用 | 透明 | $text-disabled |

### 6.5 加载与空状态

| 状态 | 表现 |
|------|------|
| 首次加载 | 资产列表显示骨架屏（3行，每行高度 140px） |
| 加载更多 | 底部显示旋转loading图标 |
| 空状态-无资产 | 空状态插图 + "暂无内容资产" + 引导文案 |
| 空状态-筛选无结果 | "未找到匹配的资产" + "尝试切换筛选条件" |
| 网络错误 | 红色提示条 + "加载失败，点击重试" |
