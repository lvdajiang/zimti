### 尺寸规格

| 区域 | 宽度 | 高度 | 内边距 |
|------|------|------|--------|
| 顶部工具栏 | 100% | 48px | 左右 $space-xl |
| 主预览区 | flex: 1 | auto (16:9) | 0 |
| 视频播放器 | 100% | 自适应(16:9) | 0 |
| 播放控制条 | 100% | 40px | 0 |
| 右侧面板 | 320px | calc(100vh - 48px - 120px - 48px) | $space-lg |
| 时间轴 | 100% | 120px | $space-md |

### 布局规则

- 预览区与右侧面板间距: $space-lg
- 预览区与时间轴间距: 0（时间轴紧贴预览区下方）
- 视频播放器保持 16:9 比例，宽度自适应
- 时间轴支持水平滚动，支持缩放（每片段显示宽度）

## 3. 字体规范

### 字体家族

- 全局字体: `"Inter", "SF Pro Display", -apple-system, BlinkMacSystemFont, "Segoe UI", sans-serif`

### 字号层级

| 层级 | 变量 | 应用场景 |
|------|------|---------|
| 视频标题 | $text-lg (16px) | 工具栏中的视频标题 |

| 字重变量 | 应用 |

## 6. 状态设计

### 6.1 视频渲染状态

| 状态 | 指示 | 表现 |
|------|------|------|
| 预览就绪 | 绿色"预览就绪" | 播放器可正常播放 |
| 渲染中 | 进度条 + 百分比 | 播放器显示渲染进度叠加层 |
| 渲染完成 | 绿色勾选 + 下载按钮 | 弹窗显示"渲染完成" |
| 渲染失败 | 红色提示 | "渲染失败: 错误信息" + 重试按钮 |
| 多版本生成中 | 各平台独立进度 | 逐个渲染 |

### 6.2 片段状态

| 状态 | 表现 |
|------|------|
| 正常 | 显示缩略图和时长 |
| 选中 | 高亮边框, 右侧面板显示该片段素材和参数 |
| 拖拽排序 | 半透明, 跟随鼠标 |
| 调整时长 | 左右拖拽手柄激活, 相邻片段自动适配 |
| 加载中 | 片段内显示旋转loading |
| 错误 | 红色边框 + 错误图标 |

### 6.3 播放器状态

| 状态 | 表现 |
|------|------|
| 空白 | 黑色背景 + "点击预览按钮生成视频" |
| 加载中 | 黑色背景 + 居中旋转loading |
| 播放中 | 视频播放, 控制条3秒后隐藏 |
| 暂停 | 暂停帧 + 居中播放按钮 + 控制条常驻 |
| 缓冲 | 显示缓冲指示器 |
| 结束 | 最后一帧 + 重新播放按钮 |

### 6.4 加载与空状态

| 状态 | 表现 |
|------|------|
| 首次加载 | 预览区显示骨架屏, 时间轴显示灰色条 |
| 无分镜数据 | "请先在脚本编辑器完成分镜脚本" |
| 素材缺失 | 片段显示红色警告"素材缺失" |
| 网络错误 | 红色提示条 + "加载失败，点击重试" |
