## 3. 字体规范

### 字体家族

- 全局字体: `"Inter", "SF Pro Display", -apple-system, BlinkMacSystemFont, "Segoe UI", sans-serif`

### 字号层级

| 层级 | 变量 | 应用场景 |
|------|------|---------|
| 页面标题 | $text-3xl (24px) | "人设配置" |
| 区块标题 | $text-lg (16px) | "基本信息""核心定位""语言风格"等 |
| 表单输入 | $text-md (14px) | 输入框文字、口头禅文字、预览文案 |
| 提示/说明 | $text-sm (12px) | 区块说明、表单标签、口头禅类型标题 |
| 标签文字 | $text-xs (11px) | 风格标签文字 |
| 样稿信息 | $text-sm (12px) | 样稿文件名 |

### 字重应用

| 字重变量 | 应用 |
|----------|------|
| $weight-bold (700) | 页面标题 |
| $weight-semibold (600) | 区块标题 |
| $weight-medium (500) | 表单标签、风格标签选中态、口头禅类型标题、按钮文字 |
| $weight-normal (400) | 表单输入文字、提示说明、口头禅文字、描述文字 |

### 行高

- 标题行高: 1.3
- 正文行高: 1.5
- 多行输入行高: 1.6
- 预览文案行高: 1.8

## 6. 状态设计

### 6.1 表单验证状态

### 6.2 保存状态

| 状态 | 按钮表现 | 提示 |
|------|---------|------|
| 未修改 | 保存按钮正常 | 无 |
| 有修改未保存 | 保存按钮 $brand-indigo 闪烁提示 | 页面离开时提示"未保存的更改" |
| 保存中 | 按钮 spinner + "保存中..." | 禁用按钮 |
| 保存成功 | 按钮 $status-success + "已保存" | 2秒后恢复 |
| 保存失败 | 按钮 $status-error + "保存失败" | 下方显示错误原因 |

### 6.3 预览状态

### 操作

| 操作 | 触发位置 | 交互方式 | 条件 |
|------|---------|---------|------|
| 保存配置 | 页面标题行"保存"按钮 | 保存所有配置 | 至少填写称呼和定位 |
