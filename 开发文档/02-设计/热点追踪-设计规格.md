## 7. 响应式规则

### 断点定义

| 断点 | 宽度范围 | 适用设备 |
|------|---------|---------|
| xl | >= 1280px | 标准桌面 |
| lg | 1024px ~ 1279px | 小桌面 |
| md | 768px ~ 1023px | 平板横屏 |
| sm | < 768px | 平板竖屏 / 移动端 |

### xl（>= 1280px，标准桌面）

| 属性 | 值 |
|------|-----|
| Content 内边距 | $space-3xl (32px) |
| 搜索框宽度 | 240px |
| 热点卡片 | 全宽 |

### lg（1024px ~ 1279px）

| 属性 | 值 |
|------|-----|
| Content 内边距 | $space-2xl (24px) |
| 搜索框宽度 | 200px |

### md（768px ~ 1023px）

| 属性 | 值 |
|------|-----|
| Content 内边距 | $space-xl (20px) |
| 搜索框宽度 | 180px |
| 筛选标签 | 可换行 |
| 趋势曲线 | 隐藏 |

### sm（< 768px）

| 属性 | 值 |
|------|-----|
| Content 内边距 | $space-lg (16px) |
| 筛选标签 | 可横向滚动，不换行 |
| 搜索框 | 全宽 |
| 关键词监控区 | 仅标签行，趋势曲线隐藏 |
| 结合建议 | 默认展开 |
