> 生成日期：2026-05-09

> 页面类型：桌面后台编辑器页
> 尺寸：1920x1080

---

## 1. 页面总览

- **页面目标**：运营者基于分镜脚本预览、微调和渲染导出视频，支持多平台版本自动派生
- **角色权限**：运营者
- **进入方式**：脚本编辑器"生成视频"按钮跳转（带入分镜脚本和素材数据）

## 6. 边缘场景

| 场景 | 处理方式 |
|------|---------|
| 无分镜数据进入页面 | 预览区显示引导文案，时间轴空白 |
| 片段素材缺失 | 片段红色边框+警告，渲染时提示补充素材 |
| 超长视频（>5分钟） | 时间轴自动缩放，支持手动缩放50%-200% |
| 渲染超时（>10分钟） | 提示"渲染时间较长，请稍候..."，可取消 |
| 多版本生成部分失败 | 显示失败平台错误信息，成功的平台可正常下载 |
| 快速连续拖拽片段 | 防抖处理，避免频繁重新预览 |
| 单个片段超过30秒 | 黄色警告提示"建议拆分为多个片段" |
| 生成多版本时主版本未完成 | "生成多版本"按钮禁用 |
| 返回脚本编辑器 | 弹出确认"是否保存当前调整？" |
