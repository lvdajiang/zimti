| 空状态-无视频 | 空状态插图 + "暂无爆款视频" + 引导说明 + "去采集"按钮 |
| 空状态-筛选无结果 | "未找到匹配的视频" + "尝试调整筛选条件" |
| 网络错误 | 红色提示条 + "加载失败，点击重试" |
| 滚动到底部 | "已展示全部 N 条视频" |

## 7. 响应式规则

### 断点定义

| 断点 | 宽度范围 | 适用设备 |
|------|---------|---------|
| xl | >= 1280px | 标准桌面 |
| lg | 1024px ~ 1279px | 小桌面 |
| md | 768px ~ 1023px | 平板横屏 |
| sm | < 768px | 平板竖屏 / 移动端 |

### xl（>= 1280px，标准桌面）

| 属性 | 值 |
|------|-----|
| 卡片网格列数 | 4列 |
| Content 内边距 | $space-3xl (32px) |
| 搜索框宽度 | 240px |
| 缩略图高度 | 180px |

### lg（1024px ~ 1279px）

| 属性 | 值 |
|------|-----|
| 卡片网格列数 | 3列 |
| Content 内边距 | $space-2xl (24px) |
| 搜索框宽度 | 200px |
| 缩略图高度 | 160px |

### md（768px ~ 1023px）

| 属性 | 值 |
|------|-----|
| 卡片网格列数 | 2列 |
| Content 内边距 | $space-xl (20px) |
| 搜索框宽度 | 180px |
| 缩略图高度 | 150px |
| 筛选栏 | 筛选标签可换行 |
| 排序选项 | 移至筛选栏下方 |

### sm（< 768px）

| 属性 | 值 |
|------|-----|
| 卡片网格列数 | 1列（列表布局） |
| Content 内边距 | $space-lg (16px) |
| 搜索框 | 全宽 |
| 缩略图高度 | 200px（单列可更大） |
| 筛选标签 | 可横向滚动，不换行 |
| 批量操作栏 | 固定在底部 |
| 数据指标 | 精简为2项（播放量+互动率） |
