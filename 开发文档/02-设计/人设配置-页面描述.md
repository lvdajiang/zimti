## 1. 页面总览

- **页面目标**：运营者配置个人身份、语言风格、口头禅等人设信息，供脚本编辑器的"去AI味"引擎和文案生成调用
- **角色权限**：运营者
- **进入方式**：Sidebar"人设配置"导航项；仪表盘快捷操作"人设配置"入口

## 6. 边缘场景

| 场景 | 处理方式 |
|------|---------|
| 称呼和定位为空时保存 | 表单验证失败，滚动到第一个错误字段 |
| 口头禅内容为空 | 不允许添加空口头禅 |
| 自定义标签重复 | 提示"该标签已存在" |
| 样稿文件格式不支持 | 提示"仅支持 .txt .md .mp3 .wav" |
| 样稿文件过大（>10MB） | 提示"文件不能超过 10MB" |
| 同时上传多个样稿 | 支持多文件上传 |
| 预览时未填写必填项 | 提示"请先填写称呼和一句话定位" |
| 样稿为音频 | 显示音频文件名+大小，不支持预览播放 |
| 多行文本超长 | 最大高 200px 后滚动 |
