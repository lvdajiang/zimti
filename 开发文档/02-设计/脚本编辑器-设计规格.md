> **页面类型**: 桌面后台编辑器 | **分辨率**: 1920x1080
> **布局模式**: Toolbar + 双栏 Content | **生成日期**: 2026-05-09

## 1. 布局结构

### 整体布局

```
┌─────────────────────────────────────────────────────────────┐
│  顶部工具栏                                                  │
│  高度: 48px | 背景: $bg-primary | 底边框: 1px $border-light  │
│  [← 返回]  视频类型: [▼]  口播占比: [--/70%]  预估时长: 2:30 │
│  [生成导演脚本] [去AI味检查] [生成口播] [朗读预览] →生成视频  │
├─────────────────────────────────────────────────────────────┤
│  Content（主内容区）                                          │
│  背景: $bg-secondary | 高度: calc(100vh - 48px - 48px)      │
│                                                             │
│  ┌─ 左栏：文案区 ──────┐ ┌─ 右栏：分镜时间轴 ────────────┐ │
│  │ 宽度: 40%            │ │ 宽度: 60%                      │ │
│  │                      │ │                                │ │
│  │  口播文案标题         │ │  [分镜段落 00:00-00:05]        │ │
│  │  ───────────         │ │  目的: 建立悬念                 │ │
│  │  可编辑文本区域       │ │  视觉: 航拍全景+字幕弹出        │ │
│  │                      │ │  听觉: 悬疑BGM渐入              │ │
│  │  支持实时编辑         │ │  增强: 大字幕+震动效果           │ │
│  │  去AI味高亮标注       │ │  口播: "你绝对想不到..."        │ │
│  │                      │ │  ────────────────              │ │
│  │                      │ │  [分镜段落 00:05-00:15]        │ │
│  │                      │ │  ...                           │ │
│  │                      │ │                                │ │
│  │                      │ │  [+ 添加段落]                   │ │
│  └──────────────────────┘ └────────────────────────────────┘ │
│                                                             │
├─────────────────────────────────────────────────────────────┤
│  底部面板：素材需求清单（可折叠）                              │
│  高度: 200px（展开）/ 40px（折叠）                            │
│  [素材类型] [描述] [来源建议] [已有/需新增]                    │
└─────────────────────────────────────────────────────────────┘
```

### 尺寸规格

| 区域 | 宽度 | 高度 | 内边距 |
|------|------|------|--------|
| 顶部工具栏 | 100% | 48px | 左右 $space-xl |
| Content | 100% | calc(100vh - 48px - 48px) | $space-lg |
| 左栏（文案区） | 40% | 100% | $space-xl |
| 右栏（分镜时间轴） | 60% | 100% | $space-xl |
| 底部面板（展开） | 100% | 200px | $space-lg |
| 底部面板（折叠） | 100% | 40px | 0 |

### 双栏布局规则

- 列间距: $space-lg
- 左栏: 固定40%宽度，文案编辑区占满可用高度
- 右栏: 固定60%宽度，分镜段落列表可滚动
- 底部面板: 默认折叠，点击"素材需求"标签展开

## 3. 字体规范

### 字体家族

- 全局字体: `"Inter", "SF Pro Display", -apple-system, BlinkMacSystemFont, "Segoe UI", sans-serif`
- 代码/时间轴字体: `"SF Mono", "Fira Code", "Consolas", monospace`（时间标签 00:00-00:05）
- 数字字体: `"DIN Alternate", "Inter", sans-serif`（占比数值、时长）

### 字号层级

| 层级 | 变量 | 应用场景 |
|------|------|---------|
| 区块标题 | $text-lg (16px) | "口播文案"、"分镜时间轴" |
| 正文 | $text-md (14px) | 口播文案正文、分镜指令内容 |
| 辅助文字 | $text-sm (12px) | 分镜指令内容、工具栏标签 |
| 标签文字 | $text-xs (11px) | 时间标签、指令类型标签、状态标签 |
| 时间标签 | $text-xs (11px) | "00:00-00:05"（等宽字体） |

### 字重应用

| 字重变量 | 应用 |
|----------|------|
| $weight-semibold (600) | 文案区标题、段落目的、口播占比数值 |
| $weight-medium (500) | 工具栏按钮文字、时间标签、指令类型标签 |
| $weight-normal (400) | 口播文案正文、分镜指令内容 |

### 行高

- 标题行高: 1.3
- 正文行高: 1.8（口播文案，适合阅读和编辑）
- 指令行高: 1.5
- 标签行高: 1.4

## 7. 响应式规则

### 断点定义

| 断点 | 宽度范围 | 适用设备 |
|------|---------|---------|
| xl | >= 1280px | 标准桌面 |
| lg | 1024px ~ 1279px | 小桌面 |
| md | 768px ~ 1023px | 平板横屏 |
| sm | < 768px | 平板竖屏 / 移动端 |

### xl（>= 1280px，标准桌面）

| 属性 | 值 |
|------|-----|
| 布局 | 双栏并列 |
| 左栏宽度 | 40% |
| 右栏宽度 | 60% |
| Content 内边距 | $space-lg (16px) |

### lg（1024px ~ 1279px）

| 属性 | 值 |
|------|-----|
| 布局 | 双栏并列 |
| 左栏宽度 | 38% |
| 右栏宽度 | 62% |
| Content 内边距 | $space-md (12px) |
| 工具栏按钮文字 | 仅显示图标，hover 显示文字 |

### md（768px ~ 1023px）

| 属性 | 值 |
|------|-----|
| 布局 | 上下分栏 + 底部标签切换（文案/分镜） |
| 标签栏 | 固定在 Content 顶部，高度 40px |
| 底部面板 | 默认折叠，不占用空间 |
| Content 内边距 | $space-md (12px) |

### sm（< 768px）

| 属性 | 值 |

## 附录：字段全覆盖清单

### 显示字段

| 显示字段 | 所在组件 | 展示位置 | 样式引用 |
|---------|---------|---------|---------|
| 时间区间 | 分镜段落 | 卡片头部 | $text-xs, 等宽字体, $text-tertiary |
| 段落目的 | 分镜段落 | 时间标签右侧 | $text-xs, $weight-semibold, $brand-indigo |
| 视觉指令 | 分镜段落 | 指令区 | 紫色图标 + $text-sm $text-secondary |
| 听觉指令 | 分镜段落 | 指令区 | 蓝色图标 + $text-sm $text-secondary |
| 增强指令 | 分镜段落 | 指令区 | 橙色图标 + $text-sm $text-secondary |
| 口播文本 | 分镜段落 | 口播区 | $text-sm $text-primary, 左侧蓝色边框 |
| 需要素材 | 底部面板 | 素材行 | $text-xs, 已有/需新增状态色 |
| 视频类型 | 工具栏 | 下拉选择器 | $text-md |
| 口播占比 | 工具栏 | 进度条 + 百分比 | 达标/未达标色 |
| 预估时长 | 工具栏 | 时长显示 | $text-sm $text-secondary |
| AI味问题 | 检查面板 | 问题列表 | 禁止词 $status-error / 建议 $status-warning |

### 操作

| 操作 | 触发位置 | 交互方式 | 条件 |
|------|---------|---------|------|
| 生成导演脚本 | 工具栏按钮 | 点击触发 | 需有文案内容 |
| 编辑文案 | 左栏编辑区 | 直接编辑 | 所有 |
| 调整时间轴 | 分镜段落拖拽手柄 | 拖拽排序 | 所有 |
| 编辑分镜指令 | 分镜段落指令行 | 点击编辑 | 所有 |
| 添加段落 | 右栏底部按钮 | 点击添加 | 所有 |
| 去AI味检查 | 工具栏按钮 | 点击弹出检查面板 | 需有文案 |
| 应用修改建议 | 检查面板替换按钮 | 点击替换 | 有建议时 |
| 生成口播音频 | 工具栏按钮 | 点击生成 | 需有口播文本 |
| 朗读预览 | 工具栏按钮 | 点击播放TTS | 需有口播文本 |
| 查看素材需求 | 底部面板标签 | 点击展开/折叠 | 所有 |
| 生成视频 | 工具栏主按钮 | 跳转视频预览页 | 脚本完整 + 去AI味通过 |
