## 1. 页面总览

- **页面目标**：运营者为视频生成多平台标题、描述和话题标签，进行SEO检查、评论区运营辅助和GEO优化，管理各平台发布状态
- **角色权限**：运营者
- **进入方式**：Sidebar"发布工作台"导航项；视频预览与渲染页渲染完成后跳转

## 6. 边缘场景

| 场景 | 处理方式 |
|------|---------|
| 标题超30字 | 字数显示变红，超限部分无法输入 |
| 描述超500字 | 字数显示变红，超限部分无法输入 |
| SEO评分<40时点"标记已发布" | 按钮禁用，tooltip"SEO评分过低，请优化后再发布" |
| 切换平台时文案未保存 | 自动保存当前平台文案，切换后加载目标平台文案 |
| 未确认AIGC标识就发布 | 弹出提醒"请确认已添加AI生成内容标识" |
| 快速连续点"换一批" | 按钮禁用+防抖，上次请求未完成时不可再次触发 |
| 话题标签超过平台限制 | 小红书≤10个，抖音≤5个，视频号≤10个，达到上限后"+"按钮消失 |
| 评论区运营话术不满意 | 可通过"换一批"重新生成 |
| GEO图文为空 | 显示"点击生成图文，从口播文案提取问答结构" |

## 7. 跨页面关联

| 关联页面 | 关联方式 | 传递参数 |
|---------|---------|---------|
| 仪表盘（任务中心） | Sidebar导航 | 无 |
| 视频预览与渲染 | 进入时接收数据 | video_id, video_url, script_text |
| 选题工作台 | 读取转化目标 | 读取 conversion_goal |
| 人设配置 | AI生成文案时读取 | 读取 persona_config（语言风格） |
| 数据监控面板 | 发布后跳转 | video_id（查看发布数据） |
