> 生成日期：2026-05-09
> 前置文档：视频详情（含文案）-设计规格.md / 自媒体内容生产系统-需求.md

> 尺寸：1920x1080

---

## 1. 页面总览

- **页面目标**：运营者查看单个爆款视频的完整信息，包括视频预览、互动数据、口播文案和AI爆款拆解，可将视频用作用选题素材
- **角色权限**：运营者
- **进入方式**：从爆款视频库点击视频卡片跳转，传递 video_id

## 6. 边缘场景

| 场景 | 处理方式 |
|------|---------|
| 视频无法播放（源站下架） | 播放器显示 "视频无法播放" 占位图 |
| 超长视频标题（>50字） | 正常换行显示，不截断 |
| 文案内容超长（>5000字） | 文本区最大高 500px 后滚动 |
| 文案编辑冲突（同时有AI在提取） | 禁用编辑按钮，提示"文案正在提取中" |
| AI拆解内容为空 | 拆解区显示"AI未能识别出明显的爆款结构" |
| 互动数据部分缺失 | 缺失项显示 "--" |
| 视频源文件下载失败 | 控制栏下载按钮禁用 |
| 快速连续点击AI拆解 | 防抖，只发送一次请求 |
| 编辑文案后未保存就离开 | 弹出确认"文案修改未保存，确定离开？" |
