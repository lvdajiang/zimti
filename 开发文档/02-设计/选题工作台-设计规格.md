### 尺寸规格

| 区域 | 宽度 | 高度 | 内边距 |
|------|------|------|--------|
| Header | 100% | 56px | 左右 $space-3xl |
| Content | 100% | calc(100vh - 56px) | 上下 $space-3xl，左右 $space-3xl |
| 页面标题行 | 100% | auto | 0 |
| 三栏容器 | 100% | calc(100vh - 56px - 120px) | 0 |
| 左栏 | 280px | 100% | $space-lg |
| 中栏 | flex: 1 | 100% | $space-lg |
| 右栏 | 360px | 100% | $space-lg |
| 底部操作栏 | 100% | 64px | 左右 $space-3xl |

### 三栏布局规则

- 列间距: $space-lg
- 左栏/右栏固定宽度，中栏自适应
- 右栏默认为空，选中选题后显示脚本骨架
- 三栏均支持独立滚动

## 3. 字体规范

### 字体家族

- 全局字体: `"Inter", "SF Pro Display", -apple-system, BlinkMacSystemFont, "Segoe UI", sans-serif`
- 数字字体: `"DIN Alternate", "Inter", sans-serif`（热度数值）

### 字号层级

| 层级 | 变量 | 应用场景 |
|------|------|---------|
| 页面标题 | $text-3xl (24px) | "选题工作台" |
| 选题卡片标题 | $text-lg (16px) | 选题方案标题 |

| 标签文字 | $text-xs (11px) | 类型标签、来源标签 |

### 字重应用

| 字重变量 | 应用 |
|----------|------|
| $weight-bold (700) | 页面标题 |

## 6. 状态设计

### 6.1 选题卡片状态

| 状态 | 边框 | 背景 | 操作按钮 |
|------|------|------|---------|
| 默认 | 2px $border-light | $bg-primary | "选用" 描边按钮 |
| 悬停 | 2px $border-medium | $bg-primary | "选用" 高亮 |
| 已选用 | 2px $brand-indigo | rgba(79,70,229,0.02) | "已选用" 实心按钮 + 勾选图标 |
| 已融合 | 2px dashed $brand-purple | rgba(147,51,234,0.02) | "已融合" 紫色标签 |
| 禁用 | 2px $border-light | $bg-primary | 按钮置灰 |

### 6.2 AI生成状态

| 状态 | 指示 | 表现 |
|------|------|------|
| 空闲 | 无 | 中栏显示引导文案"选择对标素材和热点，生成选题方案" |
| 生成中 | 蓝色旋转图标 + 文字 | "AI 正在分析素材，生成选题方案..."，中栏显示骨架屏 |
| 生成完成 | 绿色勾选 | 3个选题卡片淡入展示 |
| 生成失败 | 红色提示 | "生成失败，点击重试" |

### 6.3 热点匹配度指示

| 匹配度 | 颜色 | 显示 |
|--------|------|------|
| 高（>=80%） | $status-success | "匹配度 85%" |
| 中（50-79%） | $status-warning | "匹配度 65%" |
| 低（<50%） | $text-tertiary | "匹配度 30%" |

### 6.4 加载与空状态

| 状态 | 表现 |
|------|------|
| 首次加载 | 三栏均显示骨架屏 |
| 左栏无素材 | 空状态提示"请先到爆款视频库或热点追踪添加素材" |
| 中栏无选题 | 引导提示"选择对标素材和热点，生成选题方案" |
| 右栏无骨架 | 引导提示"选用选题后，此处展示脚本骨架" |
| 网络错误 | 对应区域显示红色提示 + "重试"按钮 |
