## 6. 状态设计

### 6.1 素材引用状态

| 状态 | 表现 |
|------|------|
| 未被引用 | 删除按钮可用 |
| 被引用 | 删除按钮禁用, hover 提示"该素材已被 N 个视频引用" |

### 6.2 上传状态

| 状态 | 表现 |
|------|------|
| 等待上传 | 文件名 + 文件大小 + "等待上传" |
| 上传中 | 进度条 + 百分比 |
| 上传成功 | 绿色勾选 + "上传成功" |
| 上传失败 | 红色提示 + "上传失败，点击重试" |

### 6.3 AI生成状态

| 状态 | 表现 |
|------|------|
| 等待输入 | 描述输入框可用, 生成按钮禁用 |
| 生成中 | 生成按钮loading + 进度条 + "AI 正在生成..." |
| 生成成功 | 预览图 + "添加到素材库"按钮 |
| 生成失败 | 红色提示 + "生成失败，请重试" |

### 6.4 交互状态

**素材卡片**

| 状态 | 边框 | 阴影 | 其他 |
|------|------|------|------|
| 默认 | 1px $border-light | 无 | - |
| 悬停 | 1px $border-medium | $hover-shadow | 缩略图覆盖层 + 操作按钮 |

### 6.5 加载与空状态

| 状态 | 表现 |
|------|------|
| 首次加载 | 素材网格显示骨架屏（5列，每卡片高度 220px） |
| 加载更多 | 底部显示旋转loading图标 |
| 空状态-无素材 | 空状态插图 + "暂无素材" + 引导按钮 |
| 空状态-筛选无结果 | "未找到匹配的素材" + "尝试切换筛选条件" |
| 网络错误 | 红色提示条 + "加载失败，点击重试" |
