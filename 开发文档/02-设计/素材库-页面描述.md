## 1. 页面总览

- **页面目标**：运营者统一管理所有素材（图片、视频、音乐、音效、地图动画），支持上传、AI生成、搜索、分类筛选和复用
- **角色权限**：运营者
- **进入方式**：Sidebar"素材库"导航项；仪表盘快捷操作"素材库"入口；脚本编辑器素材需求面板跳转

## 6. 边缘场景

| 场景 | 处理方式 |
|------|---------|
| 上传文件超过500MB | 拖拽区提示"文件大小超过限制（500MB）"，不允许上传 |
| 上传不支持的格式 | 提示"不支持的文件格式"，列出支持格式 |
| 删除被引用的素材 | 删除按钮禁用，hover 提示"该素材已被 {N} 个视频引用" |
| 上传同名文件 | 自动追加编号如"(1)"，不覆盖 |
| 大量素材（>1000） | 分页加载，每页 20 条，滚动到底自动加载更多 |
| AI生成超时（>60秒） | 提示"生成时间较长，请稍候..."，超时后显示"生成失败，请重试" |
| 快速连续点击上传 | 依次排队上传，不支持并行 |
| 搜索无结果 | 显示筛选无结果空状态 |
| 素材缩略图加载失败 | 显示灰色占位图+素材类型图标 |
| 外部导入Pexels搜索无结果 | 提示"未找到匹配素材，请尝试其他关键词" |
